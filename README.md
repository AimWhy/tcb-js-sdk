## 介绍
TCB提供开发应用所需服务和基础设施。tcb js SDK 让你可以在网页端使用JavaScript代码服务访问TCB的的服务。

## 安装
tcb js SDK 可以通过`tcb-js-sdk`来访问：
```bash
npm install --save tcb-js-sdk@latest
```

要在你的模块式使用模块可以
```js
var tcb = require("tcb-js-sdk");
```
或
```js
import * as tcb from "tcb-js-sdk";
```

或者使用官方的代码包
```
    <script src="">
```
